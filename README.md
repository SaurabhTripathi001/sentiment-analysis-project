# 💬 Sentiment Analysis System

## 📌 Overview
This project is a Sentiment Analysis System that classifies text into:
- Positive 😊
- Negative 😡
- Neutral 😐

Built using Python, NLP techniques, and Machine Learning.

---

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- NLTK
- Streamlit

---

## ⚙️ Features
- Text preprocessing (stopword removal, cleaning)
- TF-IDF vectorization
- Logistic Regression model
- Interactive UI using Streamlit
